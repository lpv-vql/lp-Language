## 制御
### wait
```
wait 数;
```
〇秒待つことができる。

また、`wait 条件;`で条件がtrueになるまで待つことができる。

### repeat
```
repeat 回数 do 処理 end
```
指定した回数繰り返すことができる。

また、`repeat 条件 do 処理 end`で条件がtrueになるまで繰り返すことができる。(他言語で言う`until`)

### forever
```
forever do 処理 end
```
ずっと繰り返すことができる。

### if
```
if 条件 then 真の処理 else 偽の処理 end
```
条件分岐が行える。また、`else 処理`の省略可能。`elif`も使える。

### while
```
while 条件 do 処理 end
```
条件がtrueの間繰り返すことができる。

### for
```
for 条件 in リスト do 処理 end
```
for each文。リストのところには[範囲](#範囲)も入れられる。

### stop
```
stop
```
今行われている処理を止めることができる。繰り返し処理の中に入れる。
